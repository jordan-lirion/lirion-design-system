FAQ

Is the theme customizable?
Yes. Replace the variables in the :root and .dark blocks in theme.css with the export from your shadcn/create preset.
Are the components accessible?
Components use native elements (dialog, details, inputs) with appropriate ARIA attributes and visible focus.
Is JavaScript required?
No. The only JavaScript is minimal and inline — tabs, dialog, dropdown, and slider. The accordion relies entirely on the native <details> element.

Help

How to change the font?
Update the --font-sans variable in the :root block of theme.css.
How to add a new component?
Create a new CSS file in its own folder, then add the corresponding @import in main.css.
Is dark mode included?
Yes. Add the .dark class to the <html> element to enable dark mode.
