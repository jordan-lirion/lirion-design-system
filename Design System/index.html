Accordion

Collapsible content built on the native <details> element. No JavaScript required.

Preview

Is the theme customizable?
Yes. Replace the variables in :root and .dark in theme.css with the export from your shadcn preset.
Are the components accessible?
Components use native elements with appropriate ARIA attributes and visible focus rings.
Is JavaScript required?
Only for tabs, dialog, dropdown, and toast. The accordion uses the native <details> element.

Properties

Class / VariableTypeDescription
.accordionBaseColumn flex container wrapping all items.
.accordion__itemPartNative <details>. Bottom border separates items.
.accordion__triggerPart<summary>. Shows a rotating chevron via ::after.
.accordion__contentPartBody text, visible when item is open.
[open]StateExpanded item — chevron rotates 180°.
--borderVariableItem separator color.
--muted-foregroundVariableContent text and chevron color.
